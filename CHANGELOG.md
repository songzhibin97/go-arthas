## 版本说明

### 版本号格式

版本号格式为 `MAJOR.MINOR.PATCH`：

- **MAJOR**: 不兼容的 API 更改
- **MINOR**: 向后兼容的功能新增
- **PATCH**: 向后兼容的问题修复

### 更改类型

- **新增**: 新功能
- **变更**: 现有功能的更改
- **废弃**: 即将移除的功能
- **移除**: 已移除的功能
- **修复**: 错误修复
- **安全**: 安全相关的更改
